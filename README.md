# envconsul

[![Build Status](https://travis-ci.org/reppard/puppet-envconsul.svg?branch=master)](https://travis-ci.org/reppard/puppet-envconsul)

## Usage

```shell
  include envconsul
```

or

```shell
  class { 'envconsul':
    version  => '0.5.0',
    platform => 'linux',
    arch     => 'amd64',
  }
```
